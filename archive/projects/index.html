Botonium ~ Discord Bot

Botonium is a fully-customizable and modular Discord moderation bot. It is currently a private bot as it is currently in development.

Features include moderation, logging, games, fun commands, and various fun APIs. If you'd like to help contribute to the development of Botonium, please contact me on Discord: Eerie#6560.

Helios Practice

Helios Practice was a public Minecraft practice PVP server for the Bedrock Edition variant of the game. The network reached a total of more than 9400 registered players and a peak of 32 concurrently online players.

Due to a lack of developer motivation and interest in the Minecraft Bedrock Edition community itself, as well as the notorious reason of financing, Helios Practice closed official on February 1st, 2021, marking the end of the server's 8-month period, being officially released on June 14th, 2020.

Developers: EerieDev, xMxZero

Sumo Events

This is a public PocketMine-MP sumo event plugin in which players fight with their fists in a tournament to win.

The plugin is fully customizable, with configurable player spawns, tasks between rounds, lobby world configuration, and scoreboard integration (which will be added in the future).

NoHate Anticheat

NoHate is an advanced PocketMine-MP anticheat plugin designed to combat cheaters in the game. The anticheat's development was initiated on November 23, 2020, and is currently in progress.

Currently integrated detections include autoclicker (with a consistency check), reach, and fly. The plugin is fully configurable, with custom ban violation counts, Discord webhooks, and more.

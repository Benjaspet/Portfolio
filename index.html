Languages

PHP: I taught myself the programming language a little over 3 years ago, and have evolved very much since then. I started out developing minuscule web pages, and then later began developing plugins for the PocketMine-MP API. More recently, I have been working on authentication systems for websites using mySQL and Docker on occasion.

Javascript: This is one of the programming languages I have the most experience with and am the most comfortable with, being joined by PHP and TypeScript. I've been working with JavaScript for a little over two years as of 2021, and have since used it in fullstack, frontend & backend development.

TypeScript: I've had experience with TypeScript for a few months, but considering it's a subset of JavaScript with just stricter typing and a few other differences, I'm relatively comfortable with it. I've since been working on an API application with a friend and have been working on rewriting some of my Discord bots and backend packages in the language as I'm a fan of its syntax & typing.

HTML: This is a frontend development language that I'm fairly comfortable with up to this point. I've made a good amount of websites using it as a backbone (such as this one) and have about a year of experience with it. While I do like coding in HTML, I've generally been a greater fan of backend development languages such as PHP, TypeScript, & Java.

CSS: I'm pretty comfortable with this styling language as it is paired with HTML quite frequently. I use it to style my webpages, and am also working on creating an open-source CSS styling library and publishing it to a CDN in order that other people can use it to improve the looks of their websites.

Java: My Java experience is quite vast. Initially starting with an academic class at my high school a few years back, it was one of the languages that sparked my love for programming, alongside PHP. The syntax of the language is beautiful, and have used it to develop smaller backend applications to small Spigot & Paper plugins for friends.
